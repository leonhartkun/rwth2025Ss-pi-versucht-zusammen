Detailed Description

Hauptprogramm zur Berechnung der Sichtweite über die Erdkrümmung.

Dieses Programm verwendet die Klasse Vektor, um eine einfache geometrische Simulation durchzuführen. Es wird berechnet, wie weit man von einem erhöhten Punkt aus sehen kann, bevor die Erdkrümmung die Sichtlinie unterbricht.
